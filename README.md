# TMDB
TMDB is my cource project in Web Acedemy, it was coded in Swift and my first experience in developing an app from scratch. The application works as a mobile client for The Movie DB. It follows MVVM software design pattern and build with Alamofire, SDWebImage, RealmSwift and Youube-ios-player-helper.
